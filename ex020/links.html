    • Eu posso criar links e personalizar as cores com a: e a:visited. Existem outras opções também.

    a{
    color: red;
    text-decoration: none;
    }
    a:visited{
    color: salmon;
    }
    a:hover{
    color: black;
    text-decoration: underline;

    Posso usar o pseudoelementos. O simbolo em stilos é "::". Porexemplo se eu quiser que todo link no final tenha a indicação de que é um link.

    a::after{
    content: ' [Link]';
    text-decoration: none;
    font-weight: normal;
    color: slategray;
    }

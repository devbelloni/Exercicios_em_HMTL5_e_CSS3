Exemplo de cores criados com os nomes das cores

h2 style="background-color: blue; color:white;">Exemplo de cores Posso usar cores comuns.

Exemplo de cores com hexadecimais

Também posso usar cores especiais, com números hexadecimais #0000ff é o azul. É um códigoi que representa o RGB e eu posso usar o Gimp para pegar esse código.

Exemplo de cores pelo código RGB

Posso configurar as cores pelo código RGB, colocando o rgb(red, green, blue) Por exemplo, posso usar o branco com rgb(255,255,255). O VSCode, quando eu aponto o nome da cor, ele apresenta um quadro com a possibilidade de representar qualquer cor em qualquer sistema. Inclusive posso escolher a transparência.

Exemplo de cores pelo código hsl

Posso configurar as cores pelo código hsl que configura matiz, saturação e brilho, colocando o shl(matiz, saturação, brilho) Por exemplo, posso usar o azul com hsl(240, 100%, 50%)

Exemplo de cores pelo código RGBA com transparêrncia.

Por exemplo, posso usar o azul com rgba(0, 0, 255, 0.259). Basta apontar para o nome da cor e mudar os detalhes.

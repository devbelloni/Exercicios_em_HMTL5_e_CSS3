Posso criar uma borda com border-width

O bom do devtools é que eu posso usar para treinamento.

Usando o padding como top, right, bottom e left crio um afastamento. Sempre é bom manter o sentido do relógio no padding.

Margem

Para usar margem, uso o comando margim. Dessa forma, configuro as margens do site, inclusive podendo centralizar o quadro como margin: auto. Isto fica no lugar de usar:

  • margin-top: 10px ;
  • margin-right: 10px ;
  • margin-bottom: 10px;
  • margin-left: 10px;

    Outline

    O Outline é uma borda externa. Mas dentro da margem, podendo estrapolar a margem.

    Simplificando a borda

    Posso simplificar todos os comandos border usando um só comando. border espessura estilo cor. Isto fica no lugar de:

  • border-width: 10px;
  • border-style: groove;
  • border-color: gray;

    Simplificando padding

    Posso simplificar o padding usando padding toppx rightpx bottompx leftpx.
    Assim fica padding: 10px 10px 10px 10px para escrever:

  • padding-top: 10px;
  • padding-right: 10px;
  • padding-bottom: 10px;
  • padding-left: 10px;
    Se as 4 forem iguais, posso por padding: 10px.

    Configurando block e inline

    Por padrão, o h1 é block e o link é inline. Mas posso mudar isso. Basta colocar display pra definir. display: block ou display: inline.

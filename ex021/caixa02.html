Grouping Tags

As gouping tags são as divs e spans. Sendo que div são box level e span são tags level.
Antigamente, colocava divs dentro de divs. Mas hoje em dia na versão html5, tem maneiras de fazer de forma diferente. Existem as grouping tags semanticas. Isso facilita os mecanismos de busca.
Posso usar header, main e footer. Os navegadores preferem.

Usando section, article e aside.

Dentro de uma seccion do main, devidfamente identificada pelo id, posso criar articles que são tudo caixas. Nos articles posso criar um aside que é uma anotação dentro do article. Posso por articles e dentro, por sections, ou vice versa.

Sombras

Posso criar sombras no documento. Não preciso colocar sombrar muito fortes. Impostante é deixar o cérebro do usuário perceber isso, sutilmente.
Não é interessante colocar sombras colocridas, pois as sombras são pretas. O interessante é mudar a transparência da cor da sombra.
Posso usar header, main e footer. Os navegadores preferem.
Para arredondar aws bordas, posso usar border-radius: 10px. Posso usar o dev tools para experimentar os tamanhos de bordas.
Ou fazer bordas separadas.

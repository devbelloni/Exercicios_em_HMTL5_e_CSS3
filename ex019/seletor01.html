Então eu coloco uma identificação id="principal". E isto faz com que eu identifico um h1 especialmente.

No style.css

Relação entre HTML e CSS

Tudo o que é id no HTML, é # em CSS

Tudo o que é class no HTML, é . em CSS

Regra do W3C

Somente posso usar um id para um só título. ou seja, para usar um mesmo id parta vários hs, poreciso usar outra configuração.
QUando tiver dois elementos que eu quero personalizar, eu uso o class.

Se eu não colocar o elemento na frente, o class que eu coloco fica:
.classequeeuusei
E tudo que eu colocar a claswswe, fica configurado.

Consigo colocar diversas clwsses:
